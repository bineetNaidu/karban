## Installation

Use the package manager [npm](https://npmjs.org/) to install dependecies for Karban.

```bash
cd client/
npm install
cd ..
cd server/
npm install
```

## Usage

```bash
## Server
npm run dev # for development
npm start # for production
npm run build # for production build

## Client
npm run dev # for development
npm start # for production
npm run build # for production build

```
